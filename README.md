# Smart-job-tracker
AI Automated job search.# 🚀 AI Smart Job Tracker (n8n + OpenAI)

An automated job tracking system that fetches job listings from the Adzuna API, filters them using AI, and sends relevant opportunities via email.

---

## 🔧 Tech Stack

* n8n (workflow automation)
* OpenAI API (AI filtering)
* Adzuna API (job data)
* Gmail API (notifications)

---

## ⚙️ How It Works

1. Fetch jobs using Adzuna API
2. Split job listings into individual items
3. Use AI to evaluate job relevance
4. Filter jobs using IF condition
5. Send selected jobs via email

---

## 🧠 AI Filtering Logic

The system uses an AI model to check if a job is relevant for:

* Data Analyst roles
* AI / Automation roles
* Python / SQL-based jobs

---

## 📩 Output

* Sends automated email alerts
* Includes job title, company, and application link
* Filters only relevant jobs
